# Velcro-Ball-Robots
An archive of all current work on the project, Velcro Ball Robots. Includes: CAD, code, and other supporting documents.

All work done in the 2026 Spring Semester was done by Tiffany Jiang.

CAD Link: https://cad.onshape.com/documents/bd9e054ba85f300119f9c010/w/c1b00e1508422f5774e232ef/e/6b519c9b8c01a9b46ea73b35?renderMode=0&uiState=69f8d6ca0b0b8b36537e9c83

All programming work was done in the Arduino IDE. To continue work, all folders in the libraries folder should be downloaded onto the local machine and put into the Arduino/libraries folder. VelcroBallMotor.ino is the main file that is being worked upon.
